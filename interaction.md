# 电脑选择系统交互设计

## 核心功能流程

### 1. 二维码扫描入口
- 用户扫描二维码进入系统
- 显示欢迎界面和扫描成功提示
- 自动跳转到电脑选择页面

### 2. 电脑选择页面
- 两个主要选项卡片：
  - 公司电脑选项：显示电脑图标、标题、描述
  - 个人电脑选项：显示电脑图标、标题、描述、200元补偿提示
- 点击后分别跳转到对应页面

### 3. 公司电脑须知页面
- 详细的注意事项列表
- 使用条款和规定
- 责任说明
- 确认按钮（需勾选同意）

### 4. 个人电脑确认页面
- 显示200元/月补偿详情
- 申请流程说明
- 确认按钮

## 交互细节

### 页面跳转逻辑
1. index.html (二维码扫描) → selection.html (电脑选择)
2. selection.html → company-policy.html (公司电脑须知)
3. selection.html → personal-confirm.html (个人电脑确认)
4. 每个页面都有返回和确认功能

### 用户反馈
- 按钮点击动画效果
- 页面切换过渡动画
- 成功提示信息
- 表单验证反馈

### 响应式设计
- 适配手机和桌面设备
- 触摸友好的按钮尺寸
- 清晰的文字排版
