# 电脑选择系统项目概述

## 文件结构

### HTML页面
1. **index.html** - 主页面/二维码扫描页面
   - 显示欢迎信息和二维码
   - 扫描成功后的过渡动画
   - 跳转到选择页面

2. **selection.html** - 电脑选择页面
   - 公司电脑选项卡片
   - 个人电脑选项卡片（含200元补偿提示）
   - 选择后的页面跳转

3. **company-policy.html** - 公司电脑须知页面
   - 详细的使用条款和注意事项
   - 责任说明和规定
   - 同意确认按钮

4. **personal-confirm.html** - 个人电脑确认页面
   - 200元/月补偿详情
   - 申请流程说明
   - 最终确认按钮

### 资源文件
- **resources/qr-code.png** - 二维码图像
- **main.js** - 主要JavaScript功能
- **styles.css** - 自定义样式（如需要）

## 功能模块

### 页面导航
- 二维码扫描 → 电脑选择
- 电脑选择 → 公司政策 或 个人确认
- 各页面的返回和确认功能

### 交互功能
- 按钮点击动画
- 页面切换过渡
- 表单验证
- 成功反馈提示

### 视觉效果
- 响应式设计
- 现代简约风格
- 微交互动画
- 加载状态显示

## 技术实现

### 前端技术
- HTML5 + CSS3 + JavaScript
- Tailwind CSS框架
- Anime.js动画库
- QR码生成和识别

### 设计原则
- 移动优先响应式设计
- 无障碍访问支持
- 快速加载优化
- 跨浏览器兼容性
